<h2>Planned functionality:</h2>
<ul>
    <li>Separate modules for host and worker modes</li>
    <li>More informative settings/guis for both</li>
    <li>Customizable reactions to pvp, deaths, etc.</li>
    <li>Applicable tasks divided among available workers</li>
</ul>
